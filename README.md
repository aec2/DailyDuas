  # 🌹 DailyDuas (Günlük Dualar ve Zikirler)
  
  **DailyDuas**, manevi yolculuğunuzu dijital dünyada düzenli ve estetik bir şekilde takip etmenizi sağlayan, modern teknolojilerle donatılmış şık bir PWA (Progressive Web App) uygulamasıdır.
</div>

---

## ✨ Özellikler

- **Sahîfelerim:** Dualarınızı ve zikirlerinizi tematik klasörler (Sahîfeler) halinde düzenleyin.
- **Zengin Kütüphane:** Hazır zikir ve dua listesine anında erişin.
- **Kişiselleştirme:** 
  - Farklı renk paletleri (Toprak, Gece, Klasik).
  - Ayarlanabilir sayaç stilleri ve ilerleme çubukları.
  - Arapça okunuşu gizleme/gösterme seçeneği.
- **Akıllı Takip:** Günlük ilerleme grafikleri ve seriler (streak) ile motivasyonunuzu koruyun.
- **Bulut Senkronizasyonu:** Google ile giriş yaparak verilerinizi tüm cihazlarınızda senkronize edin.
- **Animasyonlu Deneyim:** Şık bildirimler (Toasts) ve akıcı geçişlerle premium kullanıcı deneyimi.
- **Çevrimdışı Destek (PWA):** İnternetiniz olmasa bile zikirlerinize devam edin.

## 🚀 Kullanılan Teknolojiler

Bu proje, modern web geliştirme standartları ve en son teknolojiler kullanılarak inşa edilmiştir:

- **Framework:** [Angular](https://angular.dev/) (Signal tabanlı reaktif durum yönetimi).
- **Backend & Auth:** [Firebase](https://firebase.google.com/) (Firestore NoSQL veritabanı ve Google Authentication).
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (Custom UI/UX tasarımı).
- **Web App:** [PWA](https://web.dev/progressive-web-apps/) (Hızlı yükleme, çevrimdışı kullanım ve uygulama olarak yüklenebilme).
- **Icons & Fonts:** Lucide Icons, Google Fonts (Inter, Serif, Taha Naskh).
- **Build Tool:** [Vite](https://vitejs.dev/) / Angular CLI.

## 🛠️ Yerel Kurulum

Yerel ortamda çalıştırmak için şu adımları takip edebilirsiniz:

**Gereksinimler:** Node.js (v20+)

1. **Repoyu Klonlayın:**
   ```bash
   git clone https://github.com/aec2/DailyDuas.git
   cd DailyDuas
   ```

2. **Bağımlılıkları Yükleyin:**
   ```bash
   npm install
   ```

3. **Uygulamayı Başlatın:**
   ```bash
   npm run start
   ```
   Uygulama varsayılan olarak `http://localhost:4200` adresinde çalışacaktır.

## 📱 Uygulama Olarak Yükleme
Uygulama PWA destekli olduğu için tarayıcınızın adres çubuğundaki **"Uygulamayı Yükle"** butonuna basarak telefonunuza veya bilgisayarınıza bir yerel uygulama gibi kurabilirsiniz.
